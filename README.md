# Indus INC

Indus INC is a premium, high-end B2B corporate website designed for a leading wholesale supplier and exporter of agricultural commodities and civil infrastructure developer. 

## Project Overview
This project was designed with a "luxury editorial" aesthetic, intentionally breaking away from traditional, outdated B2B commodity websites. It leverages modern web technologies to deliver an immersive, cinematic experience that conveys permanence, trust, and unparalleled quality to international buyers.

## Key Features
- **Global Particle Canvas:** A custom HTML5 Canvas animation featuring golden particles that drift seamlessly in the background across the entire homepage.
- **Asymmetrical Editorial Layouts:** Magazine-style layouts in the Services and Products pages that utilize negative space and elegant typography.
- **Smooth Animations:** Custom scroll-triggered fade-ins, luxurious image hover reveals, and scale effects to create a fluid browsing experience.
- **Premium Aesthetics:** A carefully curated color palette featuring Obsidian black, Champagne gold accents, and Alabaster text.
- **Fully Responsive:** Flawlessly responsive across mobile, tablet, and high-resolution desktop viewports.

## Tech Stack
- **Framework:** Next.js 15 (App Router, React)
- **Styling:** Tailwind CSS (with custom utility classes and theme extensions)
- **Deployment:** Optimized for Vercel

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
